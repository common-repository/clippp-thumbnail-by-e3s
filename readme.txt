メディアライブラリの中から選ばれた画像からサムネイルを作成します。

== Description ==

画像に対して動画URLを設定し、画像サムネイルを表示します。画像クリックで動画がプレビュー再生されます。

== Installation ==

1. Upload `clippp-thumbnail` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Clippp thumbnail by E2S' menu in WordPress
1. Place `<?php do_action('clippp-thumbnail'); ?>` in your templates
